# Negative-One List

do subtraction, do simple

## Tsage

```json
{
    "location": "global",
    "client_id": "",
    "client_secret": "",
    "redirect_url": "",
    "refresh_token": "",
    "drive_id": "",
    "share_root": ""
}
```

run the binary
